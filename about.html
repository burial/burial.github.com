About Burial

I started playing World of Warcraft around July of 2008 as a Frost mage. I created my Shadow priest when Wrath of the Lich King was released and, while I still occasionally play my rogue or mage, it’s the only character I call my main, and the character I’ll be playing for as long as I’m playing the game.

I love programming (with my primary tool being Ruby) and open source software. You can find a list of my projects (some of which are Warcraft addons) on my GitHub page. If you need to get in touch, you can email me directly.

I quit Warcraft in April 2012. This site exists as an archive. For now. :)

About This Site

This website is the successor to dusknoir.net, a Shadow priest blog that ran from 2009 until 2011. It’s built with nanoc, a Ruby-powered site generator, and hosted on GitHub pages.

The layout makes extensive use of HTML 5, CSS 3 and JavaScript, and therefore requires a modern browser. Comments are served dynamically with Disqus; beware that default NoScript and Ghostery settings block Disqus and prevent comments from loading.

The symbol at the bottom represents kopimi. All content on this website is dedicated to the public domain. You’re free to share, edit and utilize anything you see here, even commercially or without attribution.
